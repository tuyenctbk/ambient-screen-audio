# AetherScreen Documentation

This directory contains all documentation for the AetherScreen project, covering development from planning to delivery.

## Navigation

- [Plan](plan/README.md): Implementation plans and task tracking.
- [Analysis](analysis/README.md): Technical analysis and design decisions.
- [Localization](localization/README.md): Support for 15+ languages.

- [Delivery](delivery/README.md): Release notes and deployment history.

## Documentation System Rules

1. **One Source of Truth**: Keep one source of truth for each rule or process.
2. **Docs First**: Update documentation before or alongside code changes.
3. **Traceability**: All technical decisions should be documented in `analysis/`.
